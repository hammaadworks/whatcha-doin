## Getting Started üöÄ

Ready to stop dreaming and start doing?

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/hammaadworks/whatcha-doin.git
    cd whatcha-doin
    ```
2.  **Install dependencies:**
    ```bash
    pnpm install
    ```
3.  **Set up your Environment Variables:**
    Create a `.env.local` file based on `.env.local.example`. You'll need:
    *   `NEXT_PUBLIC_SUPABASE_URL`
    *   `NEXT_PUBLIC_SUPABASE_ANON_KEY`
    *   `NEXT_PUBLIC_DEV_MODE_ENABLED` (set to `true` for development login bypass)

4.  **Run the development server:**
    ```bash

    pnpm dev
    ```

    Open [http://localhost:3000](http://localhost:3000) in your browser. The page auto-updates as you edit files.
